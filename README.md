# Guinilac-Project
Create a 3 question that you want to ask your weak AI. Then program it to answer accordingly. Name your AI as Jacob.

Commands : 
Jacob how to quit, 
Jacob quit,
quit, 
what is your name, 
Jacob what is my name?, 
call me (say name), 
Jacob who annoys me/you, 
Jacob I love you, 
Jacob what is 2 + 2, 
Jacob what does Peter say?
